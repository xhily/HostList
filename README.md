#### 01.采集数据 Python 每十分钟更新一次数据（可自由设置时间，例如每秒一次）
01. requests模块发送请求获得网页数据，

02. lxml模块解析html并清洗出自己想要的数据  

03. 本地化处理（写入到本地json文件[不是标准json格式]）

#### 02.处理并返回数据
接口文件  hotapi.php 会将本地json文件读取并按照需求返回为json格式接口
